# 🎮 Nivel33 - Retro Platformer Adventure

![Kotlin](https://img.shields.io/badge/Kotlin-7F52FF?style=for-the-badge&logo=kotlin&logoColor=white)
![Android](https://img.shields.io/badge/Android-3DDC84?style=for-the-badge&logo=android&logoColor=white)
![Jetpack Compose](https://img.shields.io/badge/Jetpack%20Compose-4285F4?style=for-the-badge&logo=jetpack-compose&logoColor=white)

**Nivel33** es un videojuego de plataformas de estilo retro desarrollado íntegramente en **Kotlin** utilizando **Jetpack Compose** como motor de renderizado UI y gráficos. El juego combina mecánicas clásicas de saltos con un sistema dinámico de portales y enemigos con IA de patrullaje.

---

## 🚀 Características Principales

- **Engine de Física Custom:** Implementación desde cero de gravedades, colisiones por tiles y aceleraciones.
- **Sistema de Mapas Tiled:** Carga dinámica de niveles mediante archivos `.tmj` y `.tmx`.
- **Mecánicas Avanzadas:** 
  - 🌀 **Teletransporte:** Portales interconectados (Punto A -> B -> C).
  - 🧩 **Bloques Interactivos:** Bloques que caen al pisarlos y bloques de ítems con animaciones.
  - 🧗 **Escalado:** Soporte para escaleras y movimiento vertical fluido.
- **Gráficos Pixel-Art:** Animaciones fluidas mediante spritesheets optimizados.

---

## 🛠️ Tecnologías Utilizadas

- **Lenguaje:** [Kotlin](https://kotlinlang.org/)
- **Interfaz y Gráficos:** [Jetpack Compose UI](https://developer.android.com/jetpack/compose)
- **Gestión de Mapas:** [Tiled Map Editor](https://www.mapeditor.org/)
- **Persistencia:** State Management nativo de Compose para un ciclo de vida de juego eficiente.
